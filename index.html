Tip Calculator

Instructions

Build a calculator that will take the subtotal of the tab. The tip can be either 15%, 18%, 20%, or 25% of the subtotal. The tipAmount is calculated as the subtotal * tip. Finally the totalBill will be the subtotal + the tipAmount.

Pseudocode

Variables

  • subtotal
  • tip
  • tipAmount
  • totalBill

Logic

  1. SET subtotal
  2. SET tip
  3. SET tipAmount == subtotal * tipAmount
  4. SET totalBill == tipAmount + subtotal
  5. RETURN totalBill
